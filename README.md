# Task 1: Java Environment Setup & First Program

## Objective
To verify Java installation and run a simple Java program using command prompt.

## Tools Used
- OpenJDK 8+
- Notepad
- Command Prompt (CMD)

## Steps Performed
1. Installed OpenJDK and set JAVA_HOME.
2. Verified Java installation using java -version and javac -version.
3. Wrote a simple Java program in Notepad.
4. Compiled the program using javac.
5. Executed the program using java command.

## Output
Program executed successfully and displayed output in terminal.
